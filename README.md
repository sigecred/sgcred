# CREATICPY Creditis

Gestión de préstamos simple y poderosa. Una Progressive Web App (PWA) para gestionar préstamos, similar a Payo PRO pero mejorada.

## Cómo Empezar

Para empezar a usar la aplicación, simplemente abre el archivo `src/index.html` en tu navegador.

### Instalar como PWA

Para instalar la aplicación como una PWA y usarla sin conexión, sigue estos pasos:

1.  Abre la aplicación en un navegador compatible (Chrome, Edge, Firefox).
2.  Busca el botón de "Instalar" en la barra de direcciones o en el menú del navegador.
3.  Sigue las instrucciones para añadir la aplicación a tu pantalla de inicio.
